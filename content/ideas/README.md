# Ideas de Contenido

Carpeta para guardar oportunidades de contenido por fecha, conectadas con los digests diarios.

## Como usar esta carpeta

1. Revisar fuentes y guardar hallazgos en `content/digests/YYYY-MM-DD.md`.
2. Extraer ideas accionables usando los filtros de `content/topics.md`.
3. Crear o actualizar el archivo del dia en `content/ideas/YYYY-MM-DD.md`.
4. Convertir las mejores ideas en videos, carruseles, emails, SMS, articulos o guiones de llamada.

## Nombre recomendado

```text
YYYY-MM-DD.md
```

## Plantilla de idea

```md
### Idea

- Fuente:
- Fecha:
- Audiencia:
- Zona:
- Insight:
- Hook:
- Formato:
- CTA:
- Oferta o recurso:
- Riesgo / disclaimer:
- Estado:
```

## Estados sugeridos

- `pendiente`
- `seleccionada`
- `en produccion`
- `publicada`
- `descartada`
