# Guiones de Ads para Propiedades

Plantillas base para anuncios de propiedades en Meta Ads. Están pensadas para reutilizarse en videos cortos tipo Reel, Story o Feed, manteniendo una estructura clara, un CTA consistente hacia WhatsApp y suficiente flexibilidad para adaptarse a cada propiedad.

## Cómo usar estas plantillas

1. Reemplazar los campos entre corchetes por la información real de la propiedad.
2. Mantener un solo ángulo principal por anuncio: inversión, comprador primario o incentivos.
3. Confirmar que cualquier número, incentivo o claim fuerte esté respaldado antes de grabar.
4. Cerrar siempre con el CTA estándar hacia WhatsApp, salvo que la campaña use otro destino.

## CTA base recomendado

Usar esta línea como cierre principal porque combina claridad de acción, canal y expectativa de respuesta:

> Haz clic acá y escríbeme por WhatsApp para enviarte todos los detalles.

## Script Final: Inversión / Airbnb

```text
[HOOK]
“Si estás buscando una propiedad en [zona] con potencial para Airbnb o renta, mira esto.”

[UBICACIÓN]
“Esta propiedad está en [zona], a solo [X] minutos de [punto clave], una ubicación estratégica para [vacacionar / atraer turistas / rentar].”

[OPORTUNIDAD]
“Lo que la hace interesante es que [beneficio principal de inversión]. Puedes aprovecharla para [renta corta / renta larga / uso mixto], según tu estrategia.”

[CARACTERÍSTICAS]
“Cuenta con [# habitaciones], [# baños], [# extras], y además incluye [electrodomésticos / upgrades / garaje / patio / persianas].”

[ENTRADA / NÚMEROS]
“Con [inicial / rango de entrada / condición de compra] podrías adquirir esta propiedad. Y además, [incentivo real, si aplica].”

[CTA]
“Si quieres más información sobre esta propiedad, haz clic acá y escríbeme por WhatsApp para enviarte todos los detalles.”
```

## Script Final: Comprador Primario

```text
[HOOK]
“Si estás buscando tu casa en [zona], esta opción te puede interesar.”

[UBICACIÓN]
“Está ubicada en [zona], cerca de [colegios / parques / comercios / vías principales / punto clave], lo que la hace muy práctica para vivir.”

[BENEFICIO DE VIDA]
“Es una propiedad pensada para quienes buscan [comodidad / espacio / buena ubicación / tranquilidad] sin alejarse de lo importante.”

[CARACTERÍSTICAS]
“Tiene [# habitaciones], [# baños], [# extras], y además cuenta con [electrodomésticos / patio / garaje / acabados / persianas].”

[ACCESIBILIDAD]
“Dependiendo de tu perfil, podrías comprarla con [inicial / apoyo / programa / incentivo], y eso puede ayudarte a dar el paso con más claridad.”

[CTA]
“Si quieres conocer precio, ubicación y todos los detalles, haz clic acá y escríbeme por WhatsApp para enviarte toda la información.”
```

## Script Final: Ayuda de Cierre / Incentivos

```text
[HOOK]
“Si quieres comprar casa en [zona] y aprovechar ayudas disponibles, presta atención a esta opción.”

[CONTEXTO]
“Esta propiedad está en [zona] y puede ser una muy buena oportunidad para quienes quieren comprar con apoyo en el proceso.”

[PROPUESTA]
“Además de [beneficio principal de la propiedad], también podrían existir ayudas para [gastos de cierre / inicial / proceso de compra], según tu caso.”

[CARACTERÍSTICAS]
“La casa cuenta con [# habitaciones], [# baños], [# extras], y ofrece [atributo fuerte de la propiedad].”

[INCENTIVO]
“Si calificas, podrías acceder a [monto o tipo de ayuda], lo que puede hacer más fácil entrar a tu próxima propiedad.”

[CTA]
“Si quieres revisar si aplicas y recibir toda la información, haz clic acá y escríbeme por WhatsApp para explicarte el proceso.”
```

## Notas de uso

- No usar claims absolutos como `la más económica` si no se pueden sostener.
- Cuando se mencionen ayudas o incentivos, usar lenguaje condicionado: `si aplicas`, `según tu perfil`, `podrías calificar`.
- Si el video apunta a leads por WhatsApp, conviene que el guion diga desde el cierre qué información recibirá la persona al hacer clic.
- Estas plantillas funcionan mejor si el gancho principal aparece en los primeros dos segundos del video.
