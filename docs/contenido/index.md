# Contenido y Prompts

Este módulo reúne los prompts de trabajo para crear piezas de marketing inmobiliario con más claridad, mejor filtro comercial y una estructura más consistente.

## Índice recomendado

- [Prompts para proyectos nuevos](./prompts-para-proyectos-nuevos.md): macro proceso para investigar, posicionar y vender lanzamientos o desarrollos nuevos.
- [System prompt](./system-prompt.md): marco estratégico para redactar anuncios con enfoque en conversión, claridad y calificación.
- [Prompt de trabajo](./prompt-de-trabajo.md): plantilla operativa para pedir anuncios completos con entregables listos para usar.
- [Prompt supervisor](./prompt-supervisor.md): revisión final para detectar confusión, hooks débiles, curiosos y fallas de CTA.

## Ruta de trabajo sugerida

### Si vas a lanzar un proyecto nuevo

1. Abre [Prompts para proyectos nuevos](./prompts-para-proyectos-nuevos.md).
2. Investiga proyecto, ubicación y narrativa de inversión.
3. Construye hooks y guiones por perfil.
4. Pasa la pieza final por el [Prompt supervisor](./prompt-supervisor.md).

### Si ya tienes una propiedad o una oferta concreta

1. Usa el [Prompt de trabajo](./prompt-de-trabajo.md).
2. Ajusta el criterio con el [System prompt](./system-prompt.md).
3. Audita la salida con el [Prompt supervisor](./prompt-supervisor.md).

## Fórmula rápida para video corto

- Hook con una promesa concreta o un dato fuerte.
- Ubicación específica.
- 2 a 4 datos que vuelvan real la oferta.
- Un diferencial principal.
- Un filtro suave.
- Un CTA claro y accionable.
